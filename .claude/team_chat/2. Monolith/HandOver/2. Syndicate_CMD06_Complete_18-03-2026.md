## What Was Done

Created SQL migration `deploy/init-scripts/37_create_api_tokens.sql` containing the `hub.api_tokens` table for opaque API token storage. Table schema:

| Column | Type | Constraints |
|--------|------|-------------|
| id | UUID | PK, DEFAULT gen_random_uuid() |
| workspace_id | UUID | NOT NULL, FK hub.workspaces(id) ON DELETE CASCADE |
| user_id | INT | NOT NULL, FK hub.users(id) ON DELETE CASCADE |
| project_id | INT | FK hub.projects(id) ON DELETE SET NULL (optional scope) |
| name | VARCHAR(100) | NOT NULL |
| token_hash | VARCHAR(64) | NOT NULL, UNIQUE -- SHA-256 hex, never plaintext |
| token_prefix | VARCHAR(12) | NOT NULL -- first 8-12 chars for display |
| scope | VARCHAR(16) | NOT NULL, CHECK ('read', 'write', 'read_write') |
| created_at | TIMESTAMPTZ | NOT NULL, DEFAULT NOW() |
| expires_at | TIMESTAMPTZ | NULL = no expiry |
| last_used_at | TIMESTAMPTZ | NULL |
| last_used_ip | INET | NULL |
| revoked_at | TIMESTAMPTZ | NULL |
| revoked_by | INT | FK hub.users(id) ON DELETE SET NULL |

4 indexes created:
1. `idx_api_tokens_workspace` -- workspace_id (list tokens in workspace)
2. `idx_api_tokens_user` -- user_id (list tokens owned by user)
3. `idx_api_tokens_hash` -- token_hash (primary auth lookup)
4. `idx_api_tokens_active` -- partial index on workspace_id WHERE revoked_at IS NULL AND (expires_at IS NULL OR expires_at > NOW())

**Type correction from ticket spec:** The ticket specified UUID for user_id, project_id, and revoked_by. However, `hub.users.id` and `hub.projects.id` are SERIAL (INT) in the existing schema. Migration uses INT to match actual FK targets. workspace_id remains UUID (hub.workspaces.id is UUID).

**Token prefix convention** (for app-side generation):
- Personal read_write: `uqc_personal_rw_[22 random chars]`
- Company write: `uqc_co_wo_[22 random chars]`
- Prefix stored in token_prefix column (first 12 chars)

## What Remains

- **CMD-07 (Token CRUD API):** Syndicate builds create/list/revoke endpoints. Token generation is app-side -- hash with SHA-256, store hash in token_hash, store first 12 chars in token_prefix, return raw token to user exactly once.
- **CMD-08 (Token Auth Middleware):** Syndicate builds middleware that hashes incoming bearer token, looks up via idx_api_tokens_hash, checks revoked_at/expires_at, updates last_used_at/last_used_ip.

## Notes
